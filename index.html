    Introduction

    Here, you can find the collected documentation of all public data products created by the Matrix project. This includes the schemas for our KG products and the disease list.

    Classes

    Class Description
    DiseaseListEntry A disease entry in the disease list.
    MatrixDiseaseList A list of diseases.
    MatrixEdge An edge representing a relationship between two nodes in the Biolink knowledge graph.
    MatrixEdgeList A container for MatrixEdge objects.
    MatrixNode A node in the Biolink knowledge graph.
    MatrixNodeList A container for MatrixNode objects.
